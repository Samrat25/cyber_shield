## 🎯 Quick Start (5 Minutes)

### 1. Verify Installation
```bash
cybershield --version
# Should show: cybershield, version 1.0.0
```

### 2. Initialize Your Node
```bash
cybershield node init
```
✓ Creates node ID  
✓ Detects local IP  
✓ Saves configuration  

### 3. Train ML Models
```bash
cybershield ml train --advanced
```
⏱️ Takes 2-3 minutes  
✓ Trains 4 models: Isolation Forest, Autoencoder, LSTM, XGBoost  

### 4. Register on Blockchain
```bash
cybershield node register
```
✓ Pins to IPFS  
✓ Logs on Aptos blockchain  
✓ Returns CID + TX hash  

### 5. Start Monitoring
```bash
cybershield node monitor
```
✓ Real-time ML detection  
✓ Checks every 5 seconds  
✓ Auto-logs threats  

### 6. Test It (Another Terminal)
```bash
cybershield attack simulate --type cpu --intensity 8
```
✓ Simulates attack  
✓ ML detects anomaly  
✓ Evidence logged automatically

## 🔧 All Available Commands

### Node Management
```bash
cybershield node init              # Initialize node
cybershield node register          # Register on blockchain
cybershield node monitor           # Start monitoring
cybershield node monitor --p2p     # Monitor with P2P
```

### ML Management
```bash
cybershield ml train               # Train basic models
cybershield ml train --advanced    # Train full ensemble
cybershield ml info                # Show model info
cybershield ml test                # Test detection
```

### Network Management
```bash
cybershield network listen         # Start P2P server
cybershield network connect <addr> # Connect to peer
cybershield network peers          # Show peers
```

### Attack Simulation
```bash
cybershield attack simulate --type cpu       # CPU attack
cybershield attack simulate --type memory    # Memory attack
cybershield attack simulate --type network   # Network attack
cybershield attack simulate --type combined  # All attacks
cybershield attack patterns                  # Show patterns
```

### Status & Info
```bash
cybershield status                 # Show node status
cybershield version                # Show version
cybershield quickstart             # Quick guide
cybershield --help                 # Full help
```
